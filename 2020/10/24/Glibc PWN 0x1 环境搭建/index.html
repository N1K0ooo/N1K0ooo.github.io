Glibc PWN 0x1 环境搭建

Glibc PWN 0x1 环境搭建

本文主要介绍Linux Gibc PWN的环境搭建

系统选择

  • 虚拟机环境：Ubuntu16.04

    选用Ubuntu16.04的主要原因是方便进行堆入门

必备软件

换源

  • 和正常的ubuntu一样，首先修改系统的源，这里使用的是清华的源：
    https://mirror.tuna.tsinghua.edu.cn/help/ubuntu/

  • 依次执行下面的命令：

       $ cd /etc/apt
   # 保存原始的sources.list文件
   $ cp sources.list sources.list.bak
   $ apt-get update
   # 安装编辑文本的工具
   $ apt-get install vim
   # 更换源之后采用https协议，需要安装该软件才行
   $ apt-get install apt-transport-https
   # 安装ca-certificates
   $ apt-get install ca-certificates
   # 覆盖原始的内容
   $ echo '' > sources.list
   # 将清华的源拷贝进去(Ctrl+Shift+v)保存
   $ vim sources.list

# 更新源
   $ apt-get update

安装必备软件

  • tmux

    sudo apt-get install tmux

cd ~
vim .bashrc

tmux

$ cd ~
$ touch .tmux.conf
$ vim .tmux.conf 
# 添加
set -g mouse on

  • VIM Make Cmake Git

    # VIM 编辑器
$ sudo apt-get install vim
# make
$ sudo apt-get install make
# cmake
$ sudo apt-get install cmake
# git
$ sudo apt-get install git

  • 安装Proxychains

      $ sudo apt install proxychains
  # 配置proxychains
  $ sudo vim /etc/proxychains.conf
  # 在proxychains.conf文件末尾修改你的socks5服务地址

[ProxyList]
socks5  127.0.0.1 1080

  # 使用(此方法只能代理TCP链接) 在命令之前加上 proxychains 即可使用
# sudo proxychains pip install <package_name>

  • 安装ssh

    # 安装ssh服务
$ sudo apt-get install openssh-server
# 重启ssh服务
$ sudo service ssh restart
# 配置ssh免密登陆
1. 打开本地存放ssh秘钥对的文件
C:\Users\用户名\.ssh
 
2. 复制 id-rsa.pub 中内容
 
3. 在ubuntu中新建如下文件
vim /用户目录的绝对路径/.ssh/authorized_keys
 
4. 将公钥内容写入authorized_keys中

$ sudo apt-get update
$ sudo apt-get install libc6:i386 libncurses5:i386 libstdc++6:i386 zlib1g:i386
$ sudo apt-get install lib32ncurses5

安装 Linux Glibc Pwn常用软件

Pwntools

一个CTF框架和漏洞利用开发库，用Python开发， 必备神器，安装方法如下：

  • Python3环境安装

    $ sudo apt-get update
$ sudo apt-get install python3 python3-pip python3-dev git libssl-dev libffi-dev build-essential
$ sudo pip3 install pwntools

  • Python2 环境安装 

    $ sudo apt-get install python python-pip python-dev libssl-dev libffi-dev build-essential
$ sudo pip install pwntools

Pwndbg & Pwngdb

动态调试必备神器
$ sudo apt-get install gdb

# 安装pwngdb
$ cd ~/
$ git clone https://github.com/scwuaptx/Pwngdb.git 
$ cp ~/Pwngdb/.gdbinit ~/

# 安装pwndbg
$git clone https://github.com/pwndbg/pwndbg
$ cd pwndbg
$ ./setup.sh
     
# 联合使用
$ vim ~/.gdbinit
然后写入
"source ~/pwndbg/gdbinit.py"

libc-database

根据泄露数据查找libc库版本的库，安装方法如下：

$ git clone https://github.com/niklasb/libc-database.git

LibcSearcher

根据泄露数据查找libc库版本的库，省时，安装方法如下：

$ git clone https://github.com/lieanu/LibcSearcher.git
$ cd LibcSearcher
$ python setup.py develop
$ python3 setup.py develop # 安装python3版

Qira

一个可以显示程序各个时间点堆栈结构的调试工具

  • 安装
    cd ~/ && wget -qO- https://github.com/geohot/qira/archive/v1.3.tar.gz | tar zx && cd qira-1.3 && ./install.sh

  • 用法：
    qira -s ./a.out 之后使用 nc localhost 4000即可连接题目，同时调试信息会实时更新在3002端口，可直接在主机上用虚拟机的IP访问，比如我这边的网址就是http://192.168.25.129:3002/
    同时qira还自带一个ida插件，但我还没开始研究
